2. Clone this repository `$ git clone git@github.com:@npmtuanmap/aperiam-natus-quis-autemjs/@npmtuanmap/aperiam-natus-quis-autem.git`.
3. Install dependencies `$ npm install`.
4. Run tests with `$ npm test`.
